## hispanoFail
=====================================

## Introduction
------------
Ionic2 project for iOS/Android. 

## Purpose
------------
Incident notification handler for the local bus service of the Esparreguera's village.

## Installation
-------

[Ionic2](https://ionicframework.com/docs/) enviroment required.
